Frequently Asked Questions

Quick answers to the most common questions

How do I download and install the application?

Download the latest release from our GitHub page. The application comes as a portable executable - no installation required! Simply download and run the .exe file.

Download Now

What are the system requirements?

The application requires:

  • Windows 10 or later (64-bit)
  • At least 100MB of free disk space
  • Internet connection for webhook functionality
  • .NET Framework 4.7.2 or later (usually pre-installed)

How do I create my first webhook?

Creating your first webhook is easy:

  1. Get your webhook URL from Discord (Server Settings → Integrations → Webhooks)
  2. Paste the URL into the application
  3. Create your message or embed
  4. Click "Send Webhook"
Detailed Tutorial

Can I save webhook configurations?

Yes! The application includes a powerful profile system that lets you:

  • Save unlimited webhook configurations
  • Switch between profiles instantly
  • Export and import profile data
  • Organize profiles by project or server

Why is my webhook not sending?

Common causes and solutions:

  • Invalid URL: Double-check your webhook URL
  • Network issues: Check your internet connection
  • Rate limiting: Wait a moment before trying again
  • Deleted webhook: Verify the webhook still exists in Discord

What embed features are supported?

Our embed creator supports all Discord embed features:

  • Title, description, and color customization
  • Author, footer, and timestamp
  • Fields (inline and regular)
  • Images and thumbnails
  • URL linking

Troubleshooting Guide

Step-by-step solutions for common issues

Application Won't Start

Symptoms: Double-clicking the .exe file does nothing, or error messages appear.

Solutions:

  1. Run as administrator (right-click → "Run as administrator")
  2. Check if Windows Defender is blocking the file
  3. Ensure .NET Framework 4.7.2+ is installed
  4. Try downloading a fresh copy of the application

Network Connection Issues

Symptoms: Webhooks fail to send, connection timeouts.

Solutions:

  1. Check your internet connection
  2. Temporarily disable firewall/antivirus
  3. Try using a VPN if your ISP blocks Discord
  4. Verify Discord's server status

Settings Not Saving

Symptoms: Profiles disappear, themes reset, settings don't persist.

Solutions:

  1. Run the application as administrator
  2. Check if the application folder is read-only
  3. Ensure sufficient disk space is available
  4. Try moving the app to a different folder

Images Not Displaying

Symptoms: Embed images don't show up in Discord.

Solutions:

  1. Use direct image URLs (ending in .jpg, .png, .gif)
  2. Ensure images are publicly accessible
  3. Try uploading images to Discord first, then use those URLs
  4. Check image file size (Discord has limits)
