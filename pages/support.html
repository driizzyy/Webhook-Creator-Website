üîß Common Issues

Quick solutions to frequently encountered problems

Webhook URL Invalid Error

Problem: Getting "Invalid webhook URL" error when trying to send messages.

Solution:

  1. Verify the webhook URL is correct and complete
  2. Check that the webhook hasn't been deleted from Discord
  3. Ensure you have proper permissions in the Discord server
  4. Try creating a new webhook and using that URL

Rate Limit Exceeded

Problem: Messages failing to send due to rate limiting.

Solution:

  1. Wait a few seconds before sending another message
  2. Discord limits webhooks to 5 requests per second
  3. Use the built-in queue system for batch sending
  4. Consider using multiple webhooks for high-volume use

Embed Not Displaying Correctly

Problem: Rich embeds not appearing as expected in Discord.

Solution:

  1. Check that all required fields are filled out
  2. Verify color values are valid hex codes or decimal numbers
  3. Ensure image URLs are publicly accessible
  4. Use the preview feature to test before sending

Antivirus False Positive

Problem: Antivirus software flagging the application as suspicious.

Solution:

  1. Add the application to your antivirus whitelist
  2. Download only from official GitHub releases
  3. Verify file integrity with provided checksums
  4. Report false positives to your antivirus vendor
