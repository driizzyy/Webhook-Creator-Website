Version 2.0.0 - Complete Rewrite

December 15, 2023 Major Update 3,456 downloads

Version 2.0 represents a complete rewrite of the application with a modern interface, improved performance, and many new features based on user feedback.

New Features

  • Completely redesigned user interface with modern styling
  • Advanced embed creator with visual editor
  • Profile system for saving webhook configurations

  • Image upload and management system

Improvements

  • 75% faster application startup time
  • Improved webhook validation and error handling
  • Better Discord API compliance
  • Enhanced security measures
  • Optimized for Windows 11 compatibility
