üöÄ Quick Start Guide

1

Download the Tool

Choose your preferred method:

  • Executable (Recommended): Download Discord_Webhook_Creator_Pro.exe from our releases
  • Source Code: Clone the repository and run from Python
2

Get Your Discord Token

You'll need a Discord user token to send messages:

1. Open Discord in your browser
2. Press F12 to open Developer Tools
3. Go to Application ‚Üí Local Storage ‚Üí https://discord.com
4. Find "token" and copy the value (remove quotes)
3

Configure the Application

Launch the tool and set up your preferences:

  • Paste your Discord token in the Settings tab
  • Add the Channel ID where you want to send messages
  • Test your connection using the "Refresh" button
4

Create Your First Webhook

For rich embed messages, create a webhook:

  • Click "Create Webhook" in the Discord Messages tab
  • The webhook will be automatically created and saved
  • Select it from the dropdown to use for embeds
5

Send Your First Message

You're ready to start sending messages:

  • Choose your message type (Text, Embed, or File)
  • Write your message content
  • Click "üöÄ Send Message" to deliver it

‚ùì Frequently Asked Questions

üöÄ Getting Started

Q: Do I need to install anything to use the executable?
A: No! The executable is completely portable. Just download and run it. All dependencies are included.
Q: Where do I get a Discord token?
A: Open Discord in your browser, press F12, go to Application ‚Üí Local Storage ‚Üí discord.com, and copy the "token" value (remove the quotes).
Q: How do I find a Channel ID?
A: Enable Developer Mode in Discord settings, right-click any channel, and select "Copy ID". The Channel ID is a long number.
Q: Can I use this tool on Mac or Linux?
A: Yes! While the pre-built executable is for Windows, you can run the source code on any platform with Python 3.7+.

üîß Technical Issues

Q: Why am I getting "Authentication failed"?
A: This usually means your Discord token is incorrect or expired. Get a fresh token from Discord and update it in Settings.
Q: The webhook creation fails. What's wrong?
A: Ensure you have "Manage Webhooks" permission in the target channel. Server administrators or channel moderators typically have this permission.
Q: Messages aren't sending. How do I fix this?
A: Check your internet connection, verify the Channel ID is correct, and ensure you have permission to send messages in that channel.
Q: Windows blocks the executable. Is it safe?
A: Yes, it's completely safe. Windows may show warnings for unsigned executables. Click "More info" then "Run anyway" to proceed.

üí¨ Usage Questions

Q: What's the difference between webhooks and direct messages?
A: Webhooks provide rich embeds, custom avatars, and better formatting. Direct messages are simpler but more limited in appearance.
Q: Can I send messages to multiple channels?
A: Yes! Change the Channel ID for each message, or create multiple webhooks for different channels and switch between them.
Q: How do I use custom colors for embeds?
A: Enter any valid hex color code (like #FF0000 for red). You can use online color pickers to find the perfect shade.
Q: Can I schedule messages to send later?
A: The current version doesn't support scheduling. This feature may be added in future updates based on user feedback.

üîê Security & Privacy

Q: Is my Discord token stored securely?
A: Yes, tokens are stored locally in encrypted configuration files. They never leave your computer unless you're sending messages.
Q: Does this tool comply with Discord's Terms of Service?
A: Yes! The tool uses Discord's official Webhook API for embeds, which is fully supported and recommended by Discord.
Q: Can Discord detect that I'm using this tool?
A: The tool uses standard Discord APIs just like any other Discord application. It's completely legitimate and detectable usage.
Q: What data does the tool collect?
A: The tool doesn't collect or transmit any personal data. Everything stays on your local machine.

üöÄ Advanced Features

Q: Can I customize the webhook avatar or name?
A: Currently, webhooks use default settings. Future versions may include avatar and name customization based on user requests.
Q: How do I backup my webhooks and settings?
A: Copy the webhooks.json, config.json, and gui_settings.json files from your tool directory. Restore by placing them back.
Q: Can I use multiple Discord accounts?
A: You can change tokens in Settings to switch accounts. Multiple simultaneous accounts aren't currently supported.
Q: Is there an API or command-line version?
A: The current version is GUI-only. Command-line support could be added in future releases if there's sufficient demand.

üîß Troubleshooting Guide

üö´ Connection Issues

Problem: "No token configured" error

Solution:
  1. Go to the Settings tab
  2. Enter your Discord token in the "User Token" field
  3. Click "üíæ Save Settings"
  4. Return to Discord Messages tab and click "Refresh"

Problem: "Authentication failed" status

Solution:
  1. Get a fresh Discord token from your browser
  2. Ensure you copied the entire token without extra characters
  3. Remove any quotes or spaces from the token
  4. Save and test the connection again

Problem: "Connection failed" error

Solution:
  1. Check your internet connection
  2. Verify Discord is accessible in your browser
  3. Try disabling VPN or proxy temporarily
  4. Check if your firewall is blocking the application

üí¨ Message Sending Issues

Problem: Messages not appearing in Discord

Solution:
  1. Verify the Channel ID is correct (long number)
  2. Ensure you have permission to send messages in that channel
  3. Check if the channel exists and you have access to it
  4. Try sending a simple text message first

Problem: Webhook creation fails

Solution:
  1. Ensure you have "Manage Webhooks" permission
  2. Check that the channel allows webhook creation
  3. Try creating the webhook manually in Discord first
  4. Verify your token has the necessary permissions

Problem: Embed colors not working

Solution:
  1. Use valid hex color format: #RRGGBB (e.g., #FF0000)
  2. Include the # symbol at the beginning
  3. Use 6 characters for the color code
  4. Try common colors like #5865F2 (Discord purple)

üíª Application Issues

Problem: Application won't start

Solution:
  1. Check if Windows blocked the executable
  2. Right-click the .exe ‚Üí Properties ‚Üí Unblock
  3. Run as administrator if needed
  4. Ensure you have sufficient disk space

Problem: Settings not saving

Solution:
  1. Ensure the application has write permissions
  2. Check if your antivirus is blocking file writes
  3. Try running the application as administrator
  4. Verify you have sufficient disk space

Problem: GUI appears broken or unresponsive

Solution:
  1. Try switching themes in the Settings tab
  2. Restart the application
  3. Check if Windows display scaling is causing issues
  4. Update your graphics drivers

üêç Python Source Issues

Problem: Import errors when running from source

Solution:
  1. Ensure Python 3.7+ is installed
  2. Install dependencies: pip install -r requirements.txt
  3. Try using a virtual environment
  4. Update pip to the latest version

Problem: PyInstaller build fails

Solution:
  1. Install PyInstaller: pip install pyinstaller
  2. Use the provided spec file: pyinstaller discord_tool.spec
  3. Check for any missing dependencies
  4. Try building in a clean virtual environment
